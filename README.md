# Python Basic Programs

This repository contains two beginner-friendly Python programs:

1. Star Pattern Program
2. Even and Odd Number Checker

## 📌 Program 1: Star Pattern

### Description
This program prints a left-aligned triangle star (`*`) pattern based on the number of rows entered by the user.

### Code Logic
- Takes the number of rows as input.
- Uses a `for` loop to print the pattern.
- Prints spaces for proper alignment and stars in each row.

### Example

**Input:**
```
Enter the number: 5
```

**Output:**
```
*
* *
* * *
* * * *
* * * * *
```

## 📌 Program 2: Even and Odd Number Checker

### Description
This program checks whether each number in a given range is Even or Odd.

### Code Logic
- Takes starting and ending numbers as input.
- Uses a `for` loop to iterate through the range.
- Uses the modulus (`%`) operator to determine if a number is even or odd.
- Prints the result for each number.

### Example

**Input:**
```
Start: 2
End: 10
```

**Output:**
```
Number 2 is Even
Number 3 is Odd
Number 4 is Even
Number 5 is Odd
Number 6 is Even
Number 7 is Odd
Number 8 is Even
Number 9 is Odd
Number 10 is Even
```
